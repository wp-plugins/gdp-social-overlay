The plugin show modal css overlay with Facebook Page Plugin, Twitter widget or Google Plus Badge

== Description ==

The plugin show css overlay that improove your fans and followers.

You can choice which social to promote on plugin settings. 

* Set your social profiles
* Choice frequency and where the modal overlay should appear
* Check Facebook SDK if your site isn't including it!

== Installation ==

1. Upload `gdp-SocialOverlay.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to settings page and put your informations
